Build the plots of the electric field and the point charges. 

   If this is the first time building the plots, add all the points and then autoscale.
   Otherwise, use the provided dimensions (likely from the graph bounds).
Parameters
dimensionsThe dimensions to plot, (top_left, bottom_right). Defaults to the range set by the point charges.
max_mag_lengthThe length of the largest magnitude arrow. Defaults to 20.0.
Exceptions
RuntimeErrorThe plot or axes are invalid.
